# NIRIKSHA Admin Dashboard — Screenshot Capture Guide for Hackathon PPT

## Overview

This guide specifies exactly **four (4) screenshots** to capture from the NIRIKSHA Admin Portal. Each screenshot tells a unique story. Collectively, they communicate the complete value of the platform in 20–30 seconds.

---

## SCREENSHOT 1 — Executive Command Center

### What to capture
The **Executive Dashboard** homepage showing the full command-center view.

### Page to open
`/admin` (default admin landing page)

### Scroll position
**No scrolling.** Show the full top-to-bottom visible viewport including:
- Dashboard header ("Executive Dashboard" + global search bar)
- 5 Hero KPIs row (Platform Health, Active Inspections, AI Decision Accuracy, Risk Exposure, Compliance Index)
- Agent Intelligence section (3 agent cards)
- Top portion of the analytics grid (Inspection Status Distribution + Risk Level Breakdown)

### Focus
The Hero KPIs are the star. Ensure they are all visible with their icon, value, and trend indicator.

### Browser settings
- **Resolution:** 1920×1080 (or maximize browser window)
- **Zoom:** 90% (to fit more content)
- **Viewport:** Ensure at minimum the KPIs + Agent cards + first two chart cards are visible

### Widgets that must be visible
- All 5 Hero KPI cards (Platform Health, Active Inspections, AI Decision Accuracy, Risk Exposure, Compliance Index)
- Agent Intelligence section header
- At least 2 of the 3 Agent cards (Risk Prioritization, Evidence Verification, Report Generation)
- Inspection Status Distribution (donut chart)
- Risk Level Breakdown (horizontal bar chart)

### Widgets to hide (if needed)
- None — this is the full overview

### Why this screenshot is valuable
Within 5 seconds, a judge understands this is an enterprise inspection intelligence platform with AI-driven operations, real-time KPIs, and multi-agent orchestration. The 5 KPIs immediately communicate breadth: health monitoring, inspection tracking, AI accuracy, risk exposure, and compliance.

### Judge's 5-second takeaway
"Enterprise-grade command center with AI agents, real-time KPIs, and risk/compliance monitoring."

### Professional caption (8–12 words)
**Executive Command Center: AI-Powered Inspection Intelligence in Real Time**

## SCREENSHOT 2 — Agentic AI Operations

### What to capture
The **Agent Intelligence Panel** showing the three autonomous AI agents with their metrics, confidence scores, and operational status.

### Page to open
`/admin` (same dashboard page — scroll to Agent Intelligence section)

### Scroll position
**Scroll** past the Hero KPIs until the "AGENT INTELLIGENCE" section header and all 3 agent cards are fully visible. The cards should be in the center/upper half of the viewport.

### Focus
- All 3 agent cards: **Risk Prioritization Agent**, **Evidence Verification Agent**, **Report Generation Agent**
- Each card must show: agent name, status badge (Active/Standby), metrics grid (Tasks, Last Run, Confidence %, Output), confidence bar, last output text
- The green "Active" indicators with pulsing dots
- The confidence scores (87%, 94%, 91%)
- The "View Details" action link at bottom of each card

### Browser settings
- **Resolution:** 1920×1080
- **Zoom:** 100% (standard)
- **Window:** Make the browser slightly narrower than full width (~1440px) so the 3 cards are well-balanced in a row

### Widgets that must be visible
- "AGENT INTELLIGENCE" section label
- All 3 agent cards fully rendered
- Each card showing: Bot icon, name, Active/Standby badge, 4 metrics, confidence bar, last output, View Details link

### Widgets to hide
- The Hero KPIs above (they've already been shown in Screenshot 1)
- Content below the agent cards (but it's fine if a bit shows)

### Why this screenshot is valuable
This is NIRIKSHA's differentiator. Most inspection platforms at most have a chatbot. NIRIKSHA has **three dedicated autonomous AI agents** — Risk Prioritization, Evidence Verification, and Report Generation — each with independent confidence scoring, task tracking, and human-in-the-loop indicators. This proves the platform is built on a genuine multi-agent architecture.

### Judge's 5-second takeaway
"Three autonomous AI agents — not a chatbot — with confidence scores, task metrics, and human oversight."

### Professional caption (8–12 words)
**Three Autonomous AI Agents: Risk, Evidence, and Report Generation**
